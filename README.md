# MineWatch

> **Category: HW-Scout**

A Minecraft mod that provides a real-time HUD overlay with player stats, scoreboard data, and API sync.

---

## Features

- Real-time HUD overlay on screen
- Scoreboard reader & parser
- API sync for external data
- Lightweight and performance-friendly

1. Download the .jar from [Releases](../../releases)
2. Place it into your .minecraft/mods/ folder
3. Launch with Forge **or** Fabric for Minecraft **1.16.5**
